Parallax effect examples

Positioning

The position: x y; parmeter can be used to change the default position of the parallax element relative to its alignment position. This sections contains three background images with the positions (-90, 70), (0, 70), (90, 70).

Alignment

It is possible to align the parallax element to different parts of the HTML element using the parameter alignment: horizontal vertical;. The default is center center, but other possible values are top, right, bottom, left. The images in this section are aligned to the center and the four corners.
