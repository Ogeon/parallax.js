Parallax README

Parallax

Type: JavaScript namespace

The Parallax object contains all the system methods necessary to use it. This is the only object one usually need to make things work.

Methods

Parallax.init()
 Initiates the parallax effects. Should be called when the page has finnished loading (window.onload or later). It can safely be called multiple times. 
Parallax.onscroll()
 Updates all parallax elements. It should be called when the user has scrolled the page (window.onscroll is prefered). 
Parallax.onresize()
 Updates all parallax elements with a new window size. It should be called when the user has resized the window (window.onresize is prefered). 
Parallax.enable(enable)
 Enables or disables the parallax effects. It will stop the initiation of the effects if called with false before Parallax.init() is called. The parallax effects are enabled as default.

Parameters:

  • enable [boolean] - pass true if the parallax effects should be used. Pass false otherwise.
 
Parallax.addElement(parallaxObject)
 Use this to programatically add more parallax elements.

Parameters:

  • parallaxObject [Any parallax object] - The object to add into the parallax system.

Examples

var enableParallax = true;

window.onload = Parallax.init;
window.onscroll = Parallax.onscroll;
window.onresize = Parallax.onresize;

myButton.onclick = function() {
	enableParallax = !enableParallax; //Toggle enableParallax
	Parallax.enable(enableParallax);
}

ParallaxSection

These are blocks tags where parallax effects are applied. These can contain other elements with different effects. The background image should be longer than the section box, contain a single object with transparent background or be tiled for a good effect.

Auto initializing conditions

HTML tags: <section>, <div>

Class name: parallax

HTML Attributes

data-speed="[float]"
[Optional]
 The speed of the parallax movement relative to the scroll speed*. Values smaller than 0 will make the movement slower than the scroll speed while values larger than 0 will make it faster. The value -1 will make the background follow the page movement like normal backgrounds do. -1 is also the default value when data-speed is omitted.

*Think camera movement.

JavaScript Methods

ParallaxSection(DOMElement)
 The constructor for the parallax element.

Parameters:

  • DOMElement [DOMObject] - The object to which the effects will be applyed.
 
setSpeed(speed)
 Changes the parallax speed of the object. See the HTML attribute data-speed above for more information.

Parameters:

  • speed [float] - The new parallax speed.

Examples

<section id="myCoolSection" class="parallax" data-speed="-.2">
<div id="myEvenCoolerDiv" class="parallax" data-speed=".4"></div>
</section>

ParallaxMap

These are blocks tags containing a Google Map to which parallax effects are applied. These can allso contain other elements with different effects.

This type uses the Google Map api, it has to be manually initialized. See the example below.

Auto initializing conditions

None

HTML Attributes

data-speed="[float]"
[Optional]
 The speed of the parallax movement relative to the scroll speed*. Values smaller than 0 will make the movement slower than the scroll speed while values larger than 0 will make it faster. The value -1 will make the map follow the page movement like normal elements do. -1 is also the default value when data-speed is omitted.

*Think camera movement.

JavaScript Methods

ParallaxMap(DOMElement, googleMap)
 The constructor for the parallax element.

Parameters:

  • DOMElement [DOMObject] - The object to which the effects will be applyed.
  • googleMap [google.maps.Map] - The google map object connected to the element.
 
setSpeed(speed)
 Changes the parallax speed of the object. See the HTML attribute data-speed above for more information.

Parameters:

  • speed [float] - The new parallax speed.

Examples

The JavaScript part:

//Google Map settings and stuff

var element = document.getElementById("myMapCanvas");
var map = new google.maps.Map(element, settings);
Parallax.addElement(new ParallaxMap(element, mapObject));
